Welcome to the SER322 Group 18 Final Project.

Our project is to create an app where people can:
  • View their current shifts at work.
  • Put an existing shift up for other employees to take.
  • Take over a shift put up by a coworker.

    • Our project is built using these technologies:
  • Web Framework: Python Flask
  • Database: PostgreSQL
  • Templating: Jinja2
  • Styling: Twitter Bootstrap
